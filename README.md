# 해커스공무원 2027 단기합격 설명회 (HTML 슬라이드)

브라우저에서 바로 열리는 단일 파일 발표 슬라이드입니다. 화면전환 모핑, 클릭 인터랙션(플로팅 버튼·딤 모달·범례 토글), 키보드/화살표 이동을 포함합니다.

## 실행 방법
- `index.html` 더블클릭 → 브라우저에서 열림 (별도 빌드·서버 불필요)
- 폰트는 Pretendard 웹폰트(CDN)를 사용하므로 인터넷 연결 시 정확한 톤으로 보입니다.

## 조작
- 이동: `←` / `→` (또는 우상단 희미한 화살표, 하단 점)
- 특정 페이지 바로 열기: 주소 끝에 `#번호` (예: `index.html#7`)
- 3페이지 플로팅 버튼 → 딤 모달 / 7페이지 범례 클릭 → 연도별 곡선 토글 / `Esc`로 모달 닫기

## 슬라이드 구성 (총 8장)
1. 표지
2. 국가직 · 지방직 · 서울시 차이
3. 거주지 제한 (+ 2027 예정 요건 모달)
4. 시험 과목 구성 (9급 강조)
5. 2027 개편 (한국사→한능검 / 지방7급 국어→PSAT)
6. 2025 출제기조 전환 (국어 · 영어)
7. 24~26 점수 분포 (정규분포 곡선)
8. 마무리

## 작업 메모(원고)
- `슬라이드구성.md`, `슬라이드구성2.md` — 슬라이드 원고
- `점수분포참고.md` — 7페이지 차트 원본 데이터
- `*.jpeg` — 디자인 톤앤매너 참고용
